# Diagrams

This directory contains key system diagrams in **Mermaid.js** format (`.mmd`).

## Structure

- **`/architecture`**: High-level C4 Model diagrams (Context, Container, Component).
- **`/processes`**: Sequence diagrams detailing complex user flows (e.g., Zapping, Pairing).
- **`/api`**: API interaction specifications.
- **`/database`**: Entity Relationship Diagrams (ERD).

## Viewing
You can view these files using the [Mermaid Live Editor](https://mermaid.live/) or by installing the generic "Mermaid" extension for VS Code.
